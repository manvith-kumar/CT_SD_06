# Ethical Myntra Scraper

Extracts public product data from Myntra with responsible practices.

## Data Collected
- Product names
- Current/discounted prices
- Ratings (when available)

## Ethical Compliance
- Respects robots.txt
- Rate-limited requests (2s delay)
- No personal data collection

## Output Options
- Console display
- CSV/JSON export
